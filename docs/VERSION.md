## 업그레이드 가이드

### v1.0.0에서 v1.1.0으로 업그레이드

1. **백업 생성**
   ```bash
   mysqldump -u username -p rotator_system > backup_v1.0.0.sql
   ```

2. **코드 업데이트**
   ```bash
   git pull origin main
   git checkout v1.1.0
   ```

3. **PHPIDS 라이브러리 설치**
   ```bash
   # PHPIDS 라이브러리 수동 설치 확인
   ls -la PHPIDS/lib/IDS/
   ```

4. **설정 파일 업데이트**
   - `src/db/db.php` 설정 확인
   - `PHPIDS/lib/IDS/Config/Config.ini.php` 설정 확인
   - 보안 체크 함수 연동 확인

5. **테스트 실행**
   ```bash
   php test_phpids.php
   ```
